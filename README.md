# HJBleDemo
宏佳电子模块示例代码

## iOS端备注
首先需要安装cocopods，然后进入iOS项目目录内执行下面的命令
```shell
	cd HJBleDemo/iOS/HJBleDemo
	pod update

```
### SDK does not contain 'libarclite' 
可以根据https://github.com/kamyarelyasi/Libarclite-Files  下载对应文件
